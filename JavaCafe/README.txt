# Java Café - Data atualização: 25/06/2024

Este é um projeto de sistema de gerenciamento de estoque, ainda em desenvolvimento. Atualmente, o código contém funcionalidades básicas para registro de produtos, vendas e acesso ao estoque. A interface gráfica de usuário (GUI) está em uma fase inicial de implementação. 

Por padrão o sistema roda um teste das funções de criação de compra e cancelamento de compra pelo terminal

Também é gerada a versão inicial da GUI com a execução do programa.

Na classe main.java é possível observar as rotinas implementadas.
